# Lcc
Lcc框架

## V1.0

**简介**

[Lcc](https://github.com/404Lcc/Lcc)是基于Unity的轻量级框架

## 主要功能

**热更新(可选)XLua、ILRuntime**

**UI管理**

**A星寻路**

**数据加密**

**支持TCP、Http等协议**

## 快速开始

**UI**

``` csharp
IO.panelManager.OpenPanel(PanelType.Launch);
```

**场景加载**

``` csharp
Model.IO.loadsceneManager.LoadScene(SceneName.Login, null, AssetType.Scene);
```
